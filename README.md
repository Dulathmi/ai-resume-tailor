# 🚀 Tailor AI | Precision Resume Architect

**Tailor AI** is a high-performance, full-stack application designed to bridge the gap between job seekers and Applicant Tracking Systems (ATS). Using Gemini 2.5 Flash, it analyzes resumes against job descriptions in real-time to provide expert-level tailoring advice.

## ✨ Key Features

* **Neural Match Engine:** Powered by Gemini 2.5 Flash for deep semantic analysis.
* **PDF Intelligence:** Automated text extraction from uploaded resumes.
* **Glassmorphism Dashboard:** A premium, dark-themed UI with real-time analytics.
* **Secure Cloud Storage:** Persistent storage for resumes via Supabase.
* **Dynamic CRUD:** Full management of match history with real-time DB sync.

## 🛠️ Tech Stack

* **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS.
* **Backend:** Supabase (Auth, PostgreSQL, Storage).
* **AI Model:** Google Gemini 2.5 Flash API.
* **Deployment:** Vercel.
